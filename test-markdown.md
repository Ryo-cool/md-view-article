# Markdownレンダリングテスト

## 太字（bold）の前後に句読点・記号・絵文字がある場合

### 句読点のテスト
**強調**!!
(**注意**):
**太字**。
こんにちは**世界**🙂
**テナント（Tenant）**
**データベース（Database）**や**サーバー（Server）**

### その他の記号
「**重要**」
[**リンク内太字**]
{**ブレース**}
<**アングル**>

## 番号付きリストのテスト

### 正しくレンダリングされるべき形式
1. りんご
2. みかん
3. ばなな

### ネストされたリスト
1. 果物
   1. りんご
   2. みかん
2. 野菜
   1. にんじん
   2. たまねぎ

## CommonMark/GFM 基本機能テスト

### 強調
*イタリック* と **太字** と ***太字イタリック***

### リンク
[Googleへのリンク](https://google.com)

### インラインコード
`console.log('Hello World')`

### コードブロック
```javascript
function hello() {
  console.log('Hello World');
}
```

### 順序なしリスト
- 項目1
- 項目2
  - サブ項目2-1
  - サブ項目2-2
- 項目3

### テーブル
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
| D   | E   | F   |

### 引用
> これは引用文です。
> 複数行にまたがることもできます。

## 複合的なテストケース

**太字**の中に`コード`を含む: **`value`**

リストの中の太字:
1. **重要**: これは重要な項目です。
2. **注意**!! 警告メッセージ
3. こんにちは**世界**🎉
